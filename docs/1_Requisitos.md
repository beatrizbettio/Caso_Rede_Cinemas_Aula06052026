## Requisitos Funcionais (RF)
RF01 — Cadastro de ***SESSÃO*** - O sistema deve permitir cadastrar uma nova sessão (cinema, filme, data, horário, sala).

RF02 — Registro do ***PÚBLICO*** de uma sessão - O sistema deve permitir o registro de público de cada sessão.

RF03 — Listagem de ***FILMES*** em cartaz - O sistema deve permitir a listagem dos filmes em cartaz por cinema.

## Regras de Negócio (RN)
RN01 — O público registrado não pode ser negativo.

RN02 — A sessão só pode ser cadastrada se cinema e filme existirem.

RN03 — A capacidade do cinema deve ser maior que zero.
